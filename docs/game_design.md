# 🚀 太空探险记忆翻翻乐 - 游戏设计文档

## 一、项目概述

### 1.1 游戏定位
- **目标用户**：5-10岁儿童
- **游戏类型**：教育类记忆游戏
- **核心玩法**：太空主题的卡片配对游戏
- **教育目标**：提升记忆力、认知能力和专注力

### 1.2 游戏背景
玩家作为一名小小太空探险家，通过完成不同星球上的记忆配对任务，收集星星能量，探索浩瀚宇宙。

### 1.3 游戏特色
1. **🎮 创新玩法**
   - 三个难度递进的探索关卡
   - 单词与图像配对的创新玩法
   - 直观的卡片翻转动画
   - 实时的游戏进度显示

2. **👶 儿童友好设计**
   - 简单明了的操作方式
   - 大字体清晰的界面
   - 生动有趣的表情符号
   - 鼓励性的反馈机制
   - 适合儿童认知的难度曲线

3. **🤖 智能辅助系统**
   - Wordy机器人全程陪伴和引导
   - 动态表情反馈
   - 智能提醒功能
   - 个性化鼓励文案

## 二、游戏系统

### 2.1 核心玩法循环
1. **探索阶段**
   - 玩家进入新的星球（关卡）
   - 观察并记忆卡片位置
   - 尝试匹配卡片对

2. **反馈循环**
   - 成功匹配 → 获得正向反馈 → 继续探索
   - 匹配失败 → 获得提示 → 重新尝试

3. **进阶机制**
   - 完成当前星球 → 解锁新星球
   - 获得星级评价 → 激励继续挑战

### 2.2 关卡设计

#### 第一关：启程星球
- **卡片数量**：4张（2对）
- **时间限制**：90秒
- **难度定位**：入门级
- **教学目标**：熟悉基本操作和游戏规则

#### 第二关：探索星球
- **卡片数量**：6张（3对）
- **时间限制**：90秒
- **难度定位**：进阶级
- **教学目标**：提升记忆量和判断速度

#### 第三关：挑战星球
- **卡片数量**：8张（4对）
- **时间限制**：90秒
- **难度定位**：挑战级
- **教学目标**：全面考验记忆力和专注力

### 2.3 计分系统

#### 星级评价
1. **三星评价**
   - 条件：完成时间 ≤ 90秒
   - 反馈：金色星星特效 + 特殊称号
   - 奖励：完整的彩带庆祝

2. **二星评价**
   - 条件：完成时间 ≤ 120秒
   - 反馈：银色星星特效 + 鼓励称号
   - 奖励：基础彩带庆祝

3. **一星评价**
   - 条件：完成时间 > 120秒
   - 反馈：铜色星星特效 + 鼓励称号
   - 奖励：简单庆祝效果

## 三、游戏界面设计

### 3.1 主界面布局

#### 顶部信息栏
```
┌─────────────────────────────────────┐
│  🚀 探索进度：5/9     ⏱️ 90s      │
│  [=========>------------------]     │
└─────────────────────────────────────┘
```

特点：
- 信息布局优化，左侧进度，右侧时间
- 使用进度条可视化当前进度（总计9对卡片）
- 添加动画火箭图标增加趣味性
- 流光动画提供视觉反馈
- 倒计时显示紧迫感提醒

样式规格：
- 背景：深蓝色渐变
- 文字大小：24px
- 进度条高度：20px
- 圆角：20px
- 动画效果：
  - 火箭浮动
  - 进度条流光
  - 倒计时紧急状态（≤30秒时闪烁）

进度计算：
- 第一轮：2对卡片
- 第二轮：3对卡片
- 第三轮：4对卡片
- 总计：9对卡片

时间显示：
- 初始时间：90秒
- 格式：数字 + "s"
- 紧急状态：
  - 触发条件：剩余时间 ≤ 30秒
  - 效果：红色闪烁动画
  - 提示文本：显示"加油加油，你可以的！"

### 3.2 视觉主题
- 深邃的太空背景
- 动态的星星效果
- 明亮的功能按钮
- 清晰的信息展示

### 3.3 卡片设计

#### 卡片正面
- 尺寸：120px × 160px
- 内容区：100px × 140px
- 圆角：12px
- 边框：3px立体效果
- 阴影：柔和外发光

#### 卡片背面
- 统一的太空主题图案
- 星星点缀装饰
- 光泽渐变效果
- 悬停状态微弱发光

### 3.4 动画系统

#### 1. 卡片动画
- **翻转效果**
  - 类型：3D翻转
  - 持续时间：0.3s
  - 缓动函数：ease-out
  - 触发条件：点击未翻开卡片

- **匹配失败**
  - 类型：横向晃动
  - 持续时间：0.3s
  - 晃动幅度：8px
  - 触发条件：两张卡片不匹配

#### 2. 反馈动画
- **匹配成功**
  - 类型：彩带庆祝
  - 持续时间：2s
  - 彩带数量：30个
  - 触发条件：成功配对

- **关卡完成**
  - 类型：星星评价 + 全屏庆祝
  - 持续时间：3s
  - 特效元素：彩带、星星、闪光
  - 触发条件：完成所有配对

#### 1. 状态指示动画
- **火箭浮动**
  - 类型：上下浮动
  - 持续时间：2s
  - 浮动距离：5px
  - 循环：无限循环

- **进度条流光**
  - 类型：横向流动
  - 持续时间：1.5s
  - 效果：渐变光带
  - 循环：无限循环

- **倒计时紧急**
  - 类型：颜色闪烁
  - 触发：剩余时间 ≤ 30秒
  - 颜色：#ff4757（红色）
  - 动画：1s 脉冲效果

### 3.5 提示系统

#### 1. 连续失误提示
- 触发条件：连续2次匹配失败
- 显示位置：屏幕中央
- 显示时长：2s
- 提示类型：文字 + 表情

#### 2. 超时提示
- 触发条件：15秒无操作
- 显示位置：屏幕中央
- 显示时长：3s
- 提示类型：文字 + 动画

#### 3. 时间提醒
- 触发条件：剩余30秒
- 显示位置：
  - 倒计时数字变红
  - 添加闪烁动画
- 显示效果：
  - 文字颜色：#ff4757
  - 动画：脉冲效果
- 提示类型：
  - 视觉：颜色变化 + 动画
  - 文字：紧急提示消息
