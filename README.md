# 🚀 太空探险记忆翻翻乐

一个专为5-10岁儿童设计的太空主题记忆翻翻乐游戏。通过有趣的太空探险故事情境，帮助孩子们提升记忆力和认知能力。

## ✨ 游戏特色

### 🎮 游戏玩法
- 三个难度递进的探索关卡
- 单词与图像配对的创新玩法
- 直观的卡片翻转动画
- 实时的游戏进度显示
- 智能的计时系统

### 👶 儿童友好设计
- 简单明了的操作方式
- 大字体清晰的界面
- 生动有趣的表情符号
- 鼓励性的反馈机制
- 适合儿童认知的难度曲线

### 🤖 智能辅助系统
1. **Wordy机器人助手**
   - 全程陪伴和引导
   - 动态表情反馈
   - 智能提醒功能

2. **智能提醒机制**
   - 连续失误提醒
   - 长时间未操作提示
   - 剩余时间提醒
   - 个性化鼓励文案

### 🎉 视觉反馈系统
1. **即时反馈**
   - 匹配成功时的彩带庆祝
   - 匹配失败时的晃动效果
   - 进度实时更新

2. **关卡完成反馈**
   - 星级评价系统
   - 胜利庆祝动画
   - 成就展示界面

## 🎯 游戏规则

### 基本规则
1. 每轮游戏包含不同数量的卡片对：
   - 第一轮：2对卡片
   - 第二轮：3对卡片
   - 第三轮：4对卡片

2. 玩家需要在限定时间内完成匹配：
   - 每轮90秒时间限制
   - 剩余30秒时有提醒
   - 可以随时重新开始

### 计分规则
- 完成时间 ≤ 90秒：⭐⭐⭐
- 完成时间 ≤ 120秒：⭐⭐
- 完成时间 > 120秒：⭐

## 👨‍💻 开发指南

### 本地开发
1. 克隆项目
2. 打开 `pages/game.html`
3. 使用现代浏览器运行

### 代码结构
```
project/
├── pages/
│   └── game.html    # 游戏主文件
├── assets/
│   ├── images/      # 图片资源
│   ├── sounds/      # 音效资源
```
